## LSA 潜在语义分析

通过词频统计，我们可以得到文本的特征向量，向量具有一定语义表达的能力。LSA是在此基础上挖掘文本的潜在语义，构建主题模型等。

遍历语料库，得到全部文本-单词的权重矩阵，权重即求得的TF-IDF值。建立文本-单词权重矩阵X后，可以通过SVD奇异值分解的方式将矩阵X转换为它的低秩逼近矩阵。
